ATM Management System

Using JAVA Swing and MySQL

The ATM Management System project in Java serves as a robust desktop application designed to streamline and secure banking transactions. With a user-friendly graphical interface, it caters to both administrators and users. Admins have the capability to add users to the system, while users can conduct various operations including deposits, withdrawals, pin changes, and statement inquiries. The system leverages Java's Swing module for an intuitive GUI and seamlessly integrates with a MySQL database to store and manage user data and transaction details, ensuring a smooth and efficient banking experience.

CPU Benchmarking

Using Tkinter and Python

This Python script is a cross-platform CPU benchmark tool with a Tkinter GUI. Users can configure the benchmark by setting the number of iterations and repetitions. The benchmark involves mathematical computations in nested loops, measuring execution time and calculating the average over repetitions. Information about the CPU and OS is displayed. The script is designed for Windows, macOS, and Linux compatibility, utilizing the cpuinfo library. The results, including individual run times and the average, are presented in a text box upon running the benchmark through the GUI.
